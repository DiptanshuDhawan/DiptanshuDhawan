# Diptanshu Dhawan

I work in Cybersecurity, mostly split between two things — offensive security / VAPT, and enterprise networking. I got into this because I like taking things apart to see how they break, and then building something that fixes it properly instead of patching around it.

On the certification side, I hold CCNA 200-301 (routing, switching, VLANs, ACLs, the whole enterprise networking stack) and eJPT (hands-on web and network pentesting). Both came from wanting proof, mostly to myself, that I could actually do the thing and not just talk about it.

Day to day I'm usually in Burp Suite, Nmap, Metasploit, or Wireshark on the security side, and knee-deep in TCP/IP, OSPF, or VLAN configs on the networking side. For building things, it's Python, Java, JavaScript, Bash, and Node — running on Linux, in Docker, pushed through Git.

I'm ranked in the top 3% on [TryHackMe](https://tryhackme.com/p/diptanshudhawan), and I put out security/networking content on YouTube under [HackWithDD](https://youtube.com/@HackWithDD).

## Projects

**[Revelio](https://github.com/DiptanshuDhawan/revelio)** — an AI-powered phishing/BEC detection platform. It's a Chrome extension paired with a SOC-style investigation dashboard, mixing rule-based scoring with LLM-assisted detection (OpenAI, Gemini, Ollama, or OpenRouter, with an offline fallback if none of those are available). I got to present this one at a CISO pitch event representing Chitkara University, which was a good push to make it feel like a real product instead of a class project.

**[NetLabX](https://github.com/DiptanshuDhawan/NetSimX)** — a browser-based CCNA lab environment built on GNS3, made to be a lighter, friendlier alternative to Packet Tracer. Has an in-browser terminal, automatic grading, and labs written in YAML covering routing, switching, VLANs, and ACLs.

**Linux Permission Automation CLI** — a Bash tool that finds and fixes insecure file/directory permissions. Policy-driven, and it has a dry-run mode so you can see what it would change before it actually touches anything.
